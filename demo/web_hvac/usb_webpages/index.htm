MQX Software Solutions
With the powerful integration of MCF5225x family, Freescale provides full production source code of Freescale MQX software including:

  • RTOS – Full priority-based, pre-emptive scheduler
  • Real-time TCP/IP Communication Suite (RTCS):TCP/IP, FTP, Telnet, DHCP, DNS server and client, SNMP client and more
  • USB Host/Device
  • MS-DOS File System (MFS)
  • PC Host Tools (Application Generator Tool)

Freescale MQX, a component based architecture provides scalability by allowing you to choose components required for your applications while reducing overhead.  And to make development easier, Freescale MQX RTOS task aware debugging is integrated with CodeWarrior® and IAR Embedded Workbench.

MCF5225X Family
The MCF5225x family consists of highly integrated devices with on-chip USB, Ethernet, CAN and encryption functions. The MCF5225x family is based on a 32-bit ColdFire® core with up to 80 MHz core frequency, 512 KB flash and 64 KB SRAM. The external bus interface provides flexibility to add additional memory or simple peripherals. These and other features make the MCF5225x devices ideal for industry and health care applications that require a broad range of connectivity
