
		Location of binaries for MQX in CodeWarrior 10.1


	1) MQX TAD plugin binaries
	**************************
Default <CW10_INSTALL_DIR> is C:\Program Files\Freescale\CW MCU V10.1


Source:
1) <MQX_INSTALL_PATH>\tools\codewarrior_extensions\CW MCU v10.1\mqx_p2_repository\tad_files\*.*
2) <MQX_INSTALL_PATH>\tools\codewarrior_extensions\CW MCU v10.1\mqx_p2_repository\plugins\

Target:
1) <CW10_INSTALL_DIR>\MCU\bin\plugins\Debugger\rtos\*.*
2) <CW10_INSTALL_DIR>\eclipse\plugins\





	2) MQX Project wizard for CW10.1 binaries
	***************************************

Source:
1) <MQX_INSTALL_PATH>\tools\codewarrior_extensions\CW MCU v10.1\mqx_p2_repository\plugins\
2) <MQX_INSTALL_PATH>\tools\codewarrior_extensions\CW MCU v10.1\mqx_p2_repository\wizard_files\

Target:
1) <CW10_INSTALL_DIR>\eclipse\plugins\MCU\lib\wizard_data\mqx\<MQX_VERSION>  (<MQX_VESION> is - e.g. "3.7")
2) <CW10_INSTALL_DIR>\MCU\lib\wizard_data\mqx\<MQX_VERSION>
